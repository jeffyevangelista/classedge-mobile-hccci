Change summary

Header

  • Add meta row under title (section · schedule · roster size).
  • Tap title → open class-info sheet (replaces the i icon's separate screen).

Tabs

  • Pill segmented control instead of underline.
  • Per-tab count badge.
  • Consistent with rest of app's tab UI.

Row content

  • Drop "Assignment:" / "Material:" prefix.
  • Typed icon chip per activity / file type with its own tint.
  • Compact meta line: type · due · points · views.
  • Right-aligned status pill (Released / Draft / Ungraded / Submitted ratio).

List structure

  • Section headers: Overdue / Due this week / Upcoming / Earlier.
  • Drop per-row "Overdue" pill in the Overdue group.
  • Filter chip row (status / type) above list, with counts.
  • Search bar appears when item count > 10.

Add affordance

  • Replace header + with a FAB on the active tab.
  • FAB action is tab-aware (Add lesson / Create activity).
  • RapidGrader tab has no FAB.

Empty / Error

  • Adopt the new <Fallback> for all error / empty / offline / no-results states.
