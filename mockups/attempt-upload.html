Specifics:
  • Empty state: now a dashed-border rounded-xl drop zone with a centered + icon chip ( bg-accent-soft), "Tap to add an attachment", and a format hint "Photo · PDF · Document". Reads as the primary action surface instead of a small row with an Add button on the right.
  • Filled state: rounded-xl border card with 80pt thumbnail strip on the left (image thumb or colored doc-type tile for PDF/doc). File name + meta in the body. Remove X moves into a soft danger circle. Replace becomes a full- width footer button inside the card so it visually belongs to the attachment.
  • Picking: white-translucent overlay + small spinner centered on the empty/filled card. Same size as the underlying card so layout doesn't shift.
  • Error: empty state tints amber (warning-soft) and a small "Couldn't attach. Try again." chip appears below with a warning icon. Tap target stays usable so the student can retry without an extra step.
  • All colors via theme tokens (accent, accent-soft, danger, danger-soft, warning, warning-soft, default, border, surface) — adapts to dark mode.
