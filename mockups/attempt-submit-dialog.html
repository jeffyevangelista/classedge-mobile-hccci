Notes:
  • When all questions are answered, the dialog shows a single calm green summary and the primary button reads Submit.
  • When some are unanswered, the primary button changes to Submit anyway — same affordance, but the verb acknowledges the trade-off.
  • Cancel becomes Go back — less abrupt, matches the user intent (review more, not abandon).
  • Option B jump chips: small, white-on-amber, list the actual unanswered question numbers. Tapping closes the dialog and jumps to that question.
  • Both options use warning-soft / warning theme tokens, so they adapt to dark mode automatically.
