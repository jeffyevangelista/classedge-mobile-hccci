Rules:
  • Normal (> 5 min): muted bg-default + foreground text. Calm.
  • Caution (1–5 min): amber-soft background, amber-700 text. Earns glances.
  • Critical (< 1 min): danger-soft background, danger-700 text, gentle pulse, clock swaps for a solid dot to telegraph urgency.
  • Time-up: solid danger fill, white text, X-circle icon. Locked state — pairs with the existing "Time is up! Submitting…" banner.
  • Uses font-variant-numeric: tabular-nums so digits don't jitter as they tick.
  • Same pill width across states — only colors change. Avoids layout shift each minute.
