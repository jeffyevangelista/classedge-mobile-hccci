Visual rules:
  • Per-question card: same shape as the AttemptScreen question card — rounded-xl, surface bg, border-border, 14pt padding.
  • Header row: type chip on the left (same chip used on AttemptScreen), per-question score on the right (green when full marks, amber when 0). For manually-graded types: "Manually graded" badge instead of a score.
  • MC / T-F after reveal: student's wrong pick gets a 2pt amber border + amber tint + "Your pick" badge. Correct option gets 2pt success border + soft green tint + "Correct" badge. If student got it right: single option with "Your pick · Correct" badge.
  • FillInBlank / Numeric after reveal: student's answer in a colored card (green if correct, amber if wrong). Correct answer in a second green card below — only if they were wrong.
  • Essay / Upload: always renders their own answer/file with a "Manually graded" badge. Footer line shows the grading state ("Waiting for teacher review" / "Graded by Mr. Cruz Jun 8"). No automatic correctness mark.
  • Pre-reveal: own answer rendered in an accent-tinted card with "Your pick" badge. No correctness colors anywhere. Banner at the top tells the student when reveal happens.
  • Hero: compact gradient strip with attempt # + submission date, large score (when revealed) + pass/fail pill, and a 1-line summary like "8 of 10 correct · Auto-graded".
