Shared rules:
  • Radius: rounded-xl (12pt) on all inputs.
  • Background: bg-field-background (theme token — softer than the surface) so the field reads as interactive.
  • Border: 1pt border-border at rest, 2pt border-accent when focused. Padding tightens by 1pt on focus to compensate, so the input doesn't shift.
  • Padding: 14pt vertical, 16pt horizontal. More breathing room than the 12pt today.
  • Saved indicator: small green dot + "Saved" once a value is committed (driven by the existing debounced save). On focus it reads "Editing" in muted gray.
  • Numeric specifics: larger, bold, tabular-nums value when typed — the answer is the point, so it deserves the size. Stays right-aligned.
  • Essay specifics: word + char meter stays bottom-right (where it is today), gains the saved-state dot on the left.
